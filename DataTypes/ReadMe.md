# Python Data Types

This README file provides an overview of the different data types available in Python along with code examples.

## Numeric Types

### int
Integer numbers, e.g., 5, -3, 0.

```python
x = 5
y = -3
z = 0
```
### float
Floating-point numbers, e.g., 3.14, -0.001, 2.0.

```python
pi = 3.14
temperature = -0.001
height = 2.0
```
